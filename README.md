# Doto

a slight webapp generator

### 部署流程：

#### 1. 安装Nodejs环境

#### 2. 全局安装webpack

```
sudo npm install -g webpack
```

> 目前暂时需要全局安装webpack，后续会修正为当前目录安装

#### 3. 拉取项目文件

```
git pull git@github.com:ihardcoder/Doto.git
```

#### 4. 进入项目根目录执行

```
npm install
```

安装完毕后执行

```
npm start
```

访问`localhost:3000`即可
